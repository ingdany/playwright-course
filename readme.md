# Playwright Course


> Create project 

`npm init @playwright:latest`

> Run project 

`npx playwright test`

> Show Report 

`npx playwright show-report`

> How to search xpath

```//ol[contains(@class,'ui-search-layout')]```

> Check if need upgrade

```
npm outdated @playwright/test
npm install -D @playwright/test@latest
npx playwright install 
```
